Space Defender: A 2D Particle-Based Shooter Game


Objectives
The main objectives of this project are:
● To implement graphics algorithms such as the Midpoint Circle Algorithm and Bresenham
Line Algorithm for drawing scene elements, including asteroids, ship components, and bullet
tips.
● To apply 2D transformations, specifically translation, to animate the starfield background,
falling asteroids, player bullets, and particle effects.
.
A 5
● Create a complete interactive arcade game using OpenGL and FreeGLUT, including a splash
screen, gameplay, and game-over states.
● To implement a real-time particle system generating engine thruster trails and asteroid
explosion effects
● To design a modular and well-structured OpenGL program with separate functions for each
entity type and game subsystem.
● The project aims to showcase real-time rendering by utilizing the GLUT timer function,
achieving a frame rate of approximately 60 frames per second.
